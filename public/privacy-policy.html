Política de Privacidad

Esta aplicación utiliza la API de WhatsApp Business para facilitar la comunicación entre usuarios y empresas. Nos comprometemos a proteger la privacidad de los datos de nuestros usuarios.

Datos que recopilamos

  • Número de teléfono para la comunicación vía WhatsApp.
  • Mensajes enviados y recibidos para mejorar el servicio y la experiencia del usuario.

Uso de los datos

  • Los datos se utilizan exclusivamente para brindar el servicio de mensajería y atención al cliente.
  • No compartimos ni vendemos datos personales a terceros.

Seguridad

Implementamos medidas de seguridad razonables para proteger la información de los usuarios.

Contacto

Si tienes dudas sobre esta política, puedes contactarnos a través de nuestro WhatsApp de empresa.
